## useful things

### setup.js

* following command will setup for new lesson.

```
$ ./setup.js [directory] [file1] [file2]...
```

* following command will test `setup.js`.

```
$ make test
```



## requirements

* make suer you have `node.js` version `0.8.14` or above.
* type `npm install` before get cracking.
